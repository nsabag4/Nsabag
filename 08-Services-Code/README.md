# 08-Services-Code — קוד שלושת מנועי השירות

גיבוי קוד המקור של שלושת השירותים הרצים על מחשב המשרד (`C:\dev`).
**קוד בלבד** — אין כאן `.env` אמיתי, אין הקלטות/תמונות, אין לוגים, אין `venv`/`__pycache__`, אין קבצי `state`/גיבוי.
לכל מנוע יש `.env.example`; כדי להריץ בפועל מעתיקים אותו ל-`.env` וממלאים ערכים אמיתיים. `.env` לעולם לא עולה לגיט (מוגן ב-`.gitignore`).

---

## meeting-agent — סוכן סיכומי שיחות
**מה עושה:** מאזין לתיקיית קליטה (`inbox/`, מסונכרנת מהטלפון דרך Google Drive) לקבצי הקלטות שיחה (`.m4a`),
מתמלל (Groq), מפיק סיכום ומשימות, שולח לאישור בטלגרם, וכותב ל-monday.com (חדר מיון + מנהל משימות).
**איך מריצים:** `start-agent.bat` או `python main.py`.
**מאיפה ה-.env:** העתק `.env.example` ל-`.env` ומלא: `BOT_TOKEN`, `NAHI_TELEGRAM_ID`, `GROQ_API_KEY`, `MONDAY_API_TOKEN` (שאר הערכים — מזהי לוחות — כבר מוגדרים).

## tizkoran (שומר זמן) — מנוע הזמינות
**מה עושה:** שרת FastAPI שמקבל חיישני טלפון (בלוטות' רכב, אזורים) + יומן גוגל (ICS) + עומסי תנועה + לקוחות ממתינים ממאנדיי,
ומחליט **מתי לצאת** לפגישות ו**אילו שיחות חוזרות** להמליץ בזמן פנוי → הודעות טלגרם.
**איך מריצים:** `run.bat` או `python -m uvicorn main:app --host 0.0.0.0 --port 8787`.
**מאיפה ה-.env:** העתק `.env.example` ל-`.env` ומלא: `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`, `CALENDAR_ICS_URL`, `WEBHOOK_SECRET`, ובהמשך `MONDAY_API_TOKEN`. פרטים מלאים ב-`README.md` ו-`CLAUDE.md` שבתיקייה.

## agent-gate (שער הסוכנים) — שער טלגרם לסוכנים
**מה עושה:** מאזין לכמה בוטי טלגרם במקביל (לפי `agents.json`). כל הודעה (טקסט/קול/תמונה) מנותבת ל-`claude -p` מקומי
עם פרומפט התפקיד מ-`prompts/`, ומחזירה תשובה בטקסט + קול. עונה רק לצ'אט המורשה (`ALLOWED_CHAT_ID`). כולל watchdog לשירותים.
**איך מריצים:** `run.bat` או `python main.py`.
**מאיפה ה-.env:** העתק `.env.example` ל-`.env` ומלא: `ALLOWED_CHAT_ID` (מזהה הצ'אט שלך), `BOT_TOKEN_*` לכל בוט פעיל, `MONDAY_API_TOKEN`, `GROQ_API_KEY`.
