# סוכן יומן — סבג נדל"ן

כל שיחה, הקלטה או הודעה שנשלחת לסוכן — מתומללת, מסתכמת, ונכנסת למאנדיי:
סיכום לחדר המיון (קבוצת "🎙️ סיכומי שיחות") ומשימות ללוח "מנהל משימות חכם"
(קבוצת "📥 נכנס מהסוכן"). על כל קלט מגיע אישור בטלגרם.
סיכום שמקורו בהקלטת שיחה נכתב למאנדיי **רק אחרי אישור בכפתור בטלגרם**.

## הפעלה

לחיצה כפולה על `start-agent.bat` — זהו. חלון שחור נשאר פתוח כל עוד הסוכן חי.

## הפעלה אוטומטית עם Windows

1. מקש Windows + R ← הקלד `shell:startup` ← אנטר.
2. גרור לשם קיצור דרך (קליק ימני ← Create shortcut) של `start-agent.bat`.

## בדיקת תקינות

בחלון פקודות בתיקייה הזו: `py main.py --check`
מציג ✅/❌ על כל רכיב: מפתחות, טלגרם, Groq, מאנדיי, הלוחות, ffmpeg, claude.

## איפה מה

| מה | איפה |
|---|---|
| הגדרות ומפתחות | `.env` (נוצר מ-`.env.example`) |
| יומן פעילות | `logs/` — קובץ ליום |
| קבצים שעובדו | `processed/` לפי חודש |
| קבצים שנכשלו | `failed/` + קובץ סיבה לצידם — שום דבר לא הולך לאיבוד |
| זיכרון השירות | `state.json` — לא למחוק כשהסוכן רץ |

## תקלה?

1. `py main.py --check` — לראות מה אדום.
2. להסתכל בקובץ הלוג של היום ב-`logs/`.
3. נתקעת — כתוב לקלוד: "נתקעתי בסוכן היומן, זה מה שאני רואה: ___" + צילום מסך.
