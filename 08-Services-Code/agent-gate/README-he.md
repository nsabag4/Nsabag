# שער הסוכנים (Agent Gate)

בוט טלגרם לכל אחד מארבעת הסוכנים של נחי. שאלה בטלגרם (טקסט / הקלטה קולית / תמונה)
← הסוכן בודק במקורות האמיתיים (מאנדיי, יומן, קבצי השירותים) ← תשובה בטקסט + הודעה קולית.

## הבוטים
1. **סבג מאנדיי** — `@sabag_monday_bot` — לקוחות, תיקים, לוחות
2. **סוכן פיננסי** — `@sabag_finance_bot` — כסף, תזרים, הוצאות
3. **סוכן המסכם** — `@sabag_summary_bot` — שירות סיכומי השיחות
4. **שומר זמן** — `@nahi_tizkoran_bot` ("תזכורן המזכירה") — תזכורות, רכב, אזורים

## איך זה עובד
- `main.py` מאזין לכל הבוטים במקביל (חוט לכל בוט).
- כל שאלה ← `claude -p` מקומי עם פרומפט התפקיד מ-`prompts/` והרשאות חופשיות, בתיקיית העבודה של הסוכן.
- עונה רק לצ'אט המורשה (`ALLOWED_CHAT_ID`). כל אחד אחר — נחסם בשקט.
- הקלטה קולית נכנסת ← תמלול (גרוק). תשובה עד 800 תווים ← גם הודעה קולית (gTTS + ffmpeg).
- תמונה נכנסת ← נשמרת ב-`inbox/` והסוכן מסתכל בה.
- זיכרון קצר: 3 חילופי השאלות האחרונים לכל סוכן (בזיכרון התהליך בלבד).

## כלב השמירה (watchdog)
- כל 10 דקות: בודק ששרת שומר זמן עונה (`/health`) ושסוכן המסכם רץ. נפילה ← 🚨 לטלגרם; חזרה ← ✅.
- כותב `heartbeat.txt`; שרת שומר זמן בודק את הקובץ מצדו (guard.py) ומתריע אם השער עצמו נפל.
- שים לב: אם גם השער וגם שומר זמן נפלו יחד (למשל המחשב כבוי) — אין מי שיתריע.

## הפעלה
- `run.bat` (נפתח חלון — זה השירות). עולה אוטומטית עם המחשב (קיצור בתיקיית ההפעלה: Agent Gate - Sabag).
- יומנים: `logs\agent-gate.log`. כל שאלה נרשמת: `<-` נכנס, `spawning claude`, `claude done`, `->` תשובה.
- בעליית השירות רצה בדיקה עצמית — חפש ביומן: `claude self-test rc=0 out=תקין`.

## הגדרות (.env)
- `BOT_TOKEN_*` — טוקן לכל בוט (ריק = הסוכן כבוי בשער).
- `ALLOWED_CHAT_ID` — הצ'אט היחיד שמקבל מענה.
- `CLAUDE_TIMEOUT_SEC` (ברירת מחדל 300), `WATCHDOG_SEC` (600), `VOICE_REPLIES`, `VOICE_MAX_CHARS`.
- `MONDAY_API_TOKEN` — משמש את `monday_query.py` שהסוכנים מריצים לשליפות.

## כללים למי שעובד על הקוד
1. טקסט לנחי — עברית בלבד, קצר, ממוספר.
2. אין סודות בקוד — הכול ב-.env (שמור ב-.gitignore).
3. כל לולאת האזנה עטופה כך שקריסה של הודעה אחת לא מפילה את הבוט.
4. לפני subprocess ל-claude מנקים משתני סביבה של סשן קלוד פעיל (CLAUDECODE וכו') — בלעדיהם היו תקיעות.
5. ריסטארט לשירות: להרוג גם את הפייתון, לא רק את חלון ה-cmd (סקריפט kill-gate בודק גם יתומים).
