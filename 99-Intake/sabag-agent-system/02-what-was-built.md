## מה נבנה/תוקן בסשן הזה (07–09.07)

### שומר זמן (tizkoran)
- **TomTom מחובר — עומסים חיים.** נחי נרשם ב-developer.tomtom.com (בלי כרטיס). המפתח = מפתח ברירת המחדל המלא (Db6K...GdEE). travel.py: Matrix Routing v2 (POST routing/matrix/2, routeType:fastest, traffic:live) — כי המפתח לא כלל Routing קלאסי. גיאוקוד search/2/geocode. נבדק: ירושלים→ת"א 51 דק'.
- **התראות יציאה מוסלמות מבוססות-מיקום** (נבנה 09.07, נבדק 5 תרחישים): `_still_at_origin()` לפי at_home/at_office (זיהוי Wi-Fi). הסלמה: 🔔 "תתכונן לצאת בעוד 20 דק'" (PREPARE_LEAD_MIN=20) → ⏰ "לצאת בעוד X" (lead=15) → 🚨 "אתה עדיין בבית — לצאת עכשיו" → ⚠️ "צפוי איחור" + טלפון היעד. יצא ודאי (Wi-Fi) → שקט. מיקום לא ידוע → נופל לשעון.
- **התראת איחור עם טלפון היעד** — `find_phone_for_meeting` שולף טלפון מכרטיס איש הקשר לפי שם הפגישה.
- **שיחות רכב אינטראקטיביות** — כניסה לרכב → רשימת שיחות מדורגת לפי דחיפות עם כפתור לכל שם (cbdone), הקש → סטטוס בוצע במאנדיי. תמיד נשלח (לא רק כשיש פגישה).
- **לימוד נסיעות (trip capture)** — מודד משך נסיעה אמיתי, שומר; נסיעה לא-מתועדת ביומן → מתריע.
- **הבחנת סוגי יומן** — יומן משרד (CALENDAR_OFFICE_ICS, פגישות שמגיעות אליו) / יומן חוץ (CALENDAR_TRAVEL_ICS) / online. פגישת משרד ונחי לא ברשת "10" → "פגישה במשרד, כדאי לחזור".
- **ריבוי יומנים** (CALENDAR_ICS_URLS, ממזג ומסיר כפילויות UID).
- **DEFAULT_TRAVEL_MIN=45**, LOCAL_TRAVEL_MIN=15.
- **guard.py** — בודק heartbeat של שער הסוכנים, מתריע אם נפל.
- כתובות אמיתיות: בית = האצ"ל 5 ירושלים, משרד = מונבז 3 ירושלים (תוקן מפתח תקווה). רשת משרד = "10".
