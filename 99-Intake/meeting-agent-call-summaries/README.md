## שורת מצב (נכון ל-07.07.2026 ערב)

המערכת **חיה ועובדת מקצה לקצה**: הקלטת שיחה בסמסונג ← סנכרון ענן ← תמלול ← סיכום ←
שער אישור בטלגרם ← כתיבה לחדר המיון וללוח המשימות במאנדיי. כולל: זיהוי איש קשר ותיקים,
מניעת כפילויות, סינון שיחות פרטיות, ומצב נודניק להתחייבויות של דקות ("אני חוזר אליך עוד 5 דקות").

**הערת אבטחה:** ארבעת המפתחות (טוקן בוט טלגרם, מפתח גרוק, טוקן מאנדיי, מזהה טלגרם) נמסרו בסשן
ונשמרו ב-`.env` המקומי במחשב של נחי בלבד. הערכים עצמם **הוסרו מהתיעוד הזה בכוונה** — פקודה קפואה
של המפרט: שום מפתח לא עולה לגיטהאב.
