## מטרת הפרויקט

להקים סביבת עבודה שבה **Claude Code** (ה־CLI של Anthropic) רץ על מחשב ה־Windows של נחי,
מחובר לחשבון ה־monday.com של העסק (sabag-nadlan.monday.com), כך שאפשר לתת לו משימה
משולבת מקצה לקצה:

**"חלץ נתונים מקובץ → וכשתסיים, הטמע אותם אוטומטית במערכת מאנדיי"** —
כלומר יצירת אייטמים בלוח מאנדיי מתוך נתונים שחולצו מקובץ (למשל אקסל), בפרומפט אחד,
בלי התערבות ידנית באמצע.

## מה הוקם בפועל (הושלם)

1. **Claude Code הותקן על המחשב** — גרסה 2.1.201, התקנה נייטיבית דרך
   `irm https://claude.ai/install.ps1 | iex`, במיקום `C:\Users\<user>\.local\bin\claude.exe`.
2. **תוקנה בעיית PATH** — תיקיית ההתקנה לא הייתה ב־PATH; נוספה ל־User PATH
   (דרך `[Environment]::SetEnvironmentVariable`).
3. **הושלם אשף ההגדרה הראשוני** — נבחרה ערכת נושא (Dark mode).
4. **אומת חיבור פעיל למאנדיי** — התברר ב־`claude mcp list` שקיים כבר חיבור מתארח דרך
   claude.ai connectors: `claude.ai monday.com: https://mcp.monday.com/mcp — ✓ Connected`,
   וכן חיבורים פעילים ל־Gmail, Google Drive, Google Calendar ו־Microsoft 365.
5. **נוסה גם חיבור MCP מקומי** עם טוקן API של מאנדיי
   (`claude mcp add monday -- npx -y @mondaydotcomorg/monday-api-mcp -t <TOKEN>`).
   ההוספה הצליחה טכנית (`Added stdio MCP server monday`) אבל בבדיקת חיבור התקבל
   `✗ Failed to connect` — ככל הנראה בגלל היעדר Node.js במחשב או בעיית נתיב עם שם
   משתמש בעברית. **הוחלט לוותר עליו** ולהשתמש בחיבור המתארח שכבר עובד.

## מה תוכנן וטרם הוקם / טרם בוצע

- [ ] **מחיקת החיבור המקומי הכושל** — להריץ `claude mcp remove monday` (הומלץ, טרם אושר שבוצע).
- [ ] **Regenerate לטוקן ה־API של מאנדיי** — הטוקן נחשף בצילומי מסך במהלך השיחה;
      הומלץ שוב ושוב להנפיק חדש ב־Developer Center ← API token ← Regenerate. **טרם בוצע.**
- [ ] **בדיקת קצה־לקצה** — להריץ בתוך Claude Code את הבדיקה
      "הצג לי את רשימת הלוחות שלי במאנדיי" ולוודא שהוא מחזיר את הלוחות. טרם אושר שבוצע.
- [ ] **המשימה המקורית עצמה** — חילוץ נתונים מקובץ אמיתי והטמעתם בלוח מאנדיי. טרם בוצע.
- [ ] **הוראת עבודה קבועה ב־CLAUDE.md** — הוסבר איך ליצור (`/init` בתיקיית הפרויקט ואז
      הוספת הוראה קבועה: "בכל פעם שאתה מסיים לחלץ קובץ, הטמע את הנתונים בלוח X במאנדיי").
      טרם נוצר בפועל.
- [ ] **מיפוי עמודות** — טרם הוגדר איזה קובץ מחולץ, לאיזה לוח, ואיך ממופות העמודות.

## החלטות שהתקבלו

1. **להשתמש בחיבור המתארח של claude.ai למאנדיי** (mcp.monday.com) במקום השרת המקומי
   עם npx — כי הוא כבר מחובר ועובד, והמקומי נכשל.
2. **לא לכתוב את טוקן ה־API לריפו** — הטוקן שנחשף בצילומים סומן בתיעוד זה כ־[REDACTED],
   וההמלצה המחייבת היא להנפיק טוקן חדש.
3. **דרך העבודה למשימה המשולבת**: פרומפט אחד רב־שלבי ("חלץ… וכשתסיים צור אייטמים…"),
   ולתהליך קבוע — הוראה ב־CLAUDE.md; לא נדרשת אוטומציה חיצונית.
4. **עריכת פקודות ארוכות בפנקס רשימות** לפני הדבקה ל־PowerShell — בגלל בעיות ערבוב
   כיווני טקסט (RTL/LTR) בטרמינל עם שם משתמש בעברית.
