🏛️ מערכת הסוכנים של סבג נדל"ן במאנדיי — מדריך הטמעה

הכל בעמוד אחד, בעברית, בלי היפוכים: לכל פרויקט יש כאן את הצעדים ואת הטקסטים המוכנים — כבר מחוברים ומורכבים. לוחצים 📋 העתק, מדביקים במקום שכתוב, וזהו.

סדר העבודה המומלץ

  1. מתחילים בפרויקט הקיים של המנכ"ל (סעיף 1 למטה) — החלפת Instructions + הודעת יישור קו.
  2. מקימים את שאר הפרויקטים אחד-אחד לפי הסעיפים (2–12). סדר עדיפויות אם המכסה נגמרת: מכירות → פיננסי → פרויקטים, והשאר אחרי איפוס המכסה.
  3. לסוכן האישי (סעיף 3) — קודם יוצרים במאנדיי סביבה סגורה "🔒 אישי — נחי" ומשלימים IDs.
  4. בסוף: טסט קצה-לקצה — מבקשים מהמנכ"ל "צור משימת בדיקה בבורד המשפטי", מעבירים את ההוראה לסוכן המשפטי, מחזירים את הדיווח למנכ"ל.
💡 בכל סעיף, הבלוקים ממוספרים לפי סדר ההדבקה. בלוק ה-Instructions כבר כולל בפנים את החוקה + נספחי הזיכרון הנכונים לאותו סוכן — אין צורך לחבר כלום ידנית.

1 🟦 מנכ"ל סבג-מאנדיי קיים — יישור קו
  1. היכנס לפרויקט הקיים "מנכ"ל סבג מאנדיי 360" (לא יוצרים חדש).
  2. לחץ על העיפרון ליד Instructions בפאנל הימני, מחק את הטקסט הישן שנמצא שם, והדבק את בלוק מס׳ 1 שלמטה (הוא כבר כולל את החוקה + נספח הזיכרון + זיכרון המנכ"ל). שמור.
  3. פתח שיחה חדשה (מצב Cowork) והדבק את בלוק מס׳ 2 — הודעת יישור הקו. הסוכן יעבור על הזיכרון הקיים שלו, ישמור את ההיסטוריה ויבטל רק מה שסותר.
  4. בדיקת קבלה: כתוב לו "תעדכן לי סטטוס של פריט במאנדיי". תגובה נכונה: סירוב + הפקת 🟦 הוראת ביצוע.
