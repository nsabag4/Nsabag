# סריקה עם NAPS2 — התקנה, פרופיל מומלץ ו-OCR בעברית

NAPS2‏ (Not Another PDF Scanner 2) היא תוכנת סריקה חינמית וקוד-פתוח ל-Windows. היא הדרך הנוחה ביותר לעבוד עם הסורק ביומיום: סריקת ערימת דפים ל-PDF אחד, סידור עמודים, ו-OCR שהופך את ה-PDF לניתן לחיפוש — כולל בעברית.

דרישה מקדימה: פתרון windows-bridge מותקן ועובד (בדקו שהדפדפן נפתח ב-<span dir="ltr">`http://localhost:8090`</span>). אם לא — [QUICKSTART.he.md](QUICKSTART.he.md).

## 1. התקנת NAPS2

1. הורידו את גרסת ההתקנה העדכנית ל-Windows מ-<span dir="ltr">https://www.naps2.com/download</span>.
2. נדרשת **גרסה 7.5 ומעלה** (רק ממנה קיימת הזנת כתובת ידנית לסורקי eSCL). הגרסאות הנוכחיות (8.x) מצוינות — פשוט קחו את החדשה ביותר.
3. התקינו כרגיל (הבא ← הבא ← סיום).

## 2. הוספת הסורק (חיבור eSCL ידני)

הסורק שלנו לא יופיע בחיפוש האוטומטי — זה צפוי (הוא "משודר" מתוך WSL, וגילוי אוטומטי על אותו מחשב לא אמין). מוסיפים אותו ידנית לפי כתובת:

1. פתחו NAPS2 והיכנסו לניהול פרופילי הסריקה (Profiles).
2. צרו פרופיל חדש ותנו לו שם, למשל: `סורק משרד — AV210`.
3. בבחירת ההתקן (Choose Device) בחרו בדרייבר **ESCL Driver**.
4. בחרו באפשרות הזנת כתובת ידנית (**Manual IP**) — אל תחכו לרשימת ההתקנים שמתמלאת מחיפוש ברשת.
5. הקלידו את הכתובת בדיוק כך ואשרו:

6. שמרו את הפרופיל. מעכשיו לחיצה על "סרוק" (Scan) עם הפרופיל הזה מפעילה את הסורק.

**אם שדה הכתובת מסרב לקבל את הסיומת `:8090`** (תלוי בגרסת NAPS2): עדכנו את NAPS2 לגרסה העדכנית ביותר — בגרסאות החדשות השדה מקבל כתובת עם פורט. לחלופין אפשר להזין את הכתובת המלאה <span dir="ltr">`http://localhost:8090/eSCL`</span>. שימו לב: אין להעביר את השרת לפורט 80 — השירות רץ כמשתמש לא-מורשה (saned) ואינו יכול להאזין לפורט נמוך מ-1024.

## 3. פרופיל סריקה מומלץ למסמכי משרד

בהגדרות הפרופיל קבעו:

| הגדרה | ערך מומלץ | למה |
|---|---|---|
| מקור (Paper source) | מזין הדפים (Feeder) | זהו סורק בהזנת דפים — אין לו משטח סריקה (Flatbed) |
| רזולוציה (DPI) | **300** | הסטנדרט למסמכים: חד לקריאה ול-OCR, קבצים בגודל סביר |
| מצב צבע | **צבע (Color, 24-bit)** | בטוח לכל מסמך; לחשבוניות בשחור-לבן אפשר Grayscale לקבצים קטנים יותר |
| גודל דף | A4 | ברירת המחדל המשרדית בישראל |

זרימת עבודה לערימת דפים:

1. שימו את כל הדפים במזין.
2. לחצו סרוק — כל העמודים ייכנסו לרשימה ב-NAPS2.
3. סדרו/סובבו/מחקו עמודים לפי הצורך.
4. שמרו כ-PDF (Save PDF) — קובץ אחד עם כל העמודים.

## 4. ‏OCR בעברית — ‏PDF שאפשר לחפש בו

ל-NAPS2 יש OCR מובנה (מנוע Tesseract). הפעלה חד-פעמית:

1. לחצו על כפתור ה-OCR בסרגל הכלים של NAPS2.
2. בהפעלה הראשונה NAPS2 יציע להוריד את רכיב ה-OCR ואת שפות הזיהוי — אשרו והורידו את **עברית (Hebrew)**. אם המסמכים שלכם מעורבים, הורידו גם **English**.
3. סמנו את הפעלת ה-OCR ובחרו את השפות.

מעכשיו, כששומרים PDF, יוטמע בו טקסט סמוי: אפשר לחפש מילים בעברית בתוך הקובץ (Ctrl+F בכל קורא PDF) ולהעתיק טקסט. שימו לב — OCR מאריך מעט את זמן השמירה; זה תקין.

טיפ לאיכות OCR בעברית: סרקו ב-300dpi לפחות, ודאגו שהדף נכנס ישר למזין — טקסט עקום מזוהה גרוע.
