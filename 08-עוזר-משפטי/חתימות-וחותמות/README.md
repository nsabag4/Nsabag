> כאן שמורים קובצי הדוגמה שהעוזר המשפטי מטביע במסמכים. השימוש בהם — אך ורק לפי שלושת המנעולים שבחוקה: מסמך שנבדק "תקין לחתימה" + אישור פרטני של נחי לאותו מסמך + הטבעה במקום המיועד בלבד.

## איך מעלים

- **בשיחה (הכי פשוט):** מצרפים את התמונות בצ'אט עם העוזר — והוא שומר אותן כאן בשמות הנכונים.
- **מהמחשב:** מעתיקים את הקבצים לתיקייה הזו, ואז שומרים לגיט (commit + push).

## אזהרת אבטחה

אלה קבצים רגישים — מי שמחזיק בהם יכול לחקות את חתימת עורך הדין. לכן:
1. הריפו חייב להישאר **פרטי** — לעולם לא ציבורי.
2. הקבצים לא נשלחים לאף גורם ולא עוזבים את התיקייה הזו.
3. העוזר משתמש בהם רק בתוך מסמך שאושר פרטנית, ולעולם לא מציג או משתף אותם בנפרד מהמסמך.
