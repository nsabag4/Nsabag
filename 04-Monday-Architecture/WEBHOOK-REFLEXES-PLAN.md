## המטרה

שהלוח יגיב לבד: פריט חדש או שינוי סטטוס בלוח ⇦ תוך שניות סוכן הקלוד של אותה מחלקה מתעורר, מטפל לפי המוח שלו, וכותב חזרה עדכון חתום בשמו. בלי סוכני מאנדיי, בלי קרדיטים — קלוד בלבד (ההחלטה הקפואה).

## הארכיטקטורה

```
אירוע בלוח (פריט חדש / סטטוס השתנה)
  ← אוטומציית מאנדיי: "Send webhook" (דחיפה מיידית, חינם)
  ← כתובת ציבורית של שער הסוכנים (Tailscale Funnel ← פורט 8787 הקיים)
  ← נתיב חדש בשער: /monday/webhook

  ← claude -p עם ה-AGENT.md של הסוכן + נתוני האירוע
  ← פעולה במאנדיי + עדכון חתום ("— סוכן X") + הודעת טלגרם אם קריטי
```

## כללי בטיחות (מחייבים)

1. כל סוכן פועל אך ורק לפי חוקי הברזל של ה-AGENT.md שלו: אין מחיקה, אין שינוי מבנה, אין נתונים כספיים — גם ברפלקס.
2. **כל פעולת רפלקס נחתמת:** עדכון על הפריט "— סוכן X, רפלקס אוטומטי, [תאריך שעה]". נחי תמיד רואה מי עשה מה.
3. **חלון דדופ:** אירוע כפול בתוך 60 שניות על אותו פריט ⇦ מטופל פעם אחת (תבנית mirror_state הקיימת).
4. **אימות:** מענה ל-challenge של מאנדיי + סוד משותף בכתובת — רק מאנדיי יכול להעיר את השער.
5. **כשל בשקט אסור:** רפלקס שנכשל ⇦ הודעת טלגרם לנחי דרך בוט הסוכן + רישום ביומן. כלב השמירה הקיים מנטר גם את הנתיב החדש.
6. **בלם הצפה:** מקסימום רפלקסים במקביל = 2; מעבר לזה נכנס לתור. פרץ אירועים (ייבוא המוני) ⇦ השהיה אוטומטית + דיווח.
