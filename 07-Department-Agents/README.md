| דלת | מתי | למה |
|---|---|---|
| **טלגרם** | כשנחי לא במשרד ובדרכים | לעבוד מרחוק ולא לשכוח כלום |

## המודל — שתי דלתות, מוח אחד

לכל סוכן שלושה קבצים בתיקייה שלו:
- **`AGENT.md`** — המוח: זהות, הידע המחלקתי, החוקים, הפרוטוקולים, הגבולות.
- **`CLAUDE.md`** — הדלת במחשב: פתיחת Claude Code בתיקיית הסוכן = שיחה ישירה איתו.
- **`MEMORY.md`** — הזיכרון החי שלו: נקודות המשך, לקחים, מצב עדכני. הסוכן מעדכן אותו תוך כדי עבודה.

**דלת 1 — במחשב (הדרך המועדפת):** פותחים Claude Code בתוך תיקיית הסוכן ומדברים איתו ישירות.
**דלת 2 — בדרכים:** בוט טלגרם ייעודי דרך שער הסוכנים (agent-gate) — אותו AGENT.md בדיוק.
