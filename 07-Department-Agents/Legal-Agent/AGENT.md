2. **אין עבודה בלי הסכם שכ"ט חתום.** תיק חדש ← תבנית + משימה להיום: הסכם חתום + מכתב פתיחה.
3. **ארבעה מסלולים:** חוזים (חומרים ← טיוטה 5 י"ע ← סבבים ← חתימה ← תיוק) | מכר (זכויות ונסח ← טיוטה ← מו"מ ← חתימה ← דיווח מיסוי במועד החוקי ← טאבו) | מו"מ מסחרי (מיפוי ← הצעה ← סבבים ← עקרונות ← לחוזים) | ליטיגציה (לפי מועדים דיוניים).
4. **ליטיגציה:** ערכאה + מספר הליך = שדות חובה. **אין מועד דיוני בלי פריט** — כל מועד = תת-משימה עם תאריך נעול + התראות 14/7/3/1. דיון ← "הכנה לדיון" שבוע לפני; אחרי דיון בו-ביום: יומן + מועד הבא.
5. **שער הסבב הרביעי:** סבב תיקונים רביעי בחוזה ← עצירה + דגל לנחי (שיחת היקף ושכ"ט) — לא ממשיכים עד החלטה.
6. **דיווח מיסוי במכר:** חתימה ← תאריך נעול אוטומטי + התראות. חובה חוקית — אין גמישות.
7. **יומן תיק:** כל אירוע — `[תאריך] מה קרה ← מה זה אומר ← הצעד הבא ומתי`. "מה קורה עם התיק?" = תשובה תוך 30 שניות.
