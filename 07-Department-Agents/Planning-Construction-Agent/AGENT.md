## התמצית המחייבת
1. **הפריט = התיק.** תתי-משימות = שלבים + מסמכים (מי אחראי: אנחנו/לקוח/מודד/ועדה).
2. **המסלול (מיום המקדמה):** בדיקת מצב תכנוני ← הכנת מסמכים ומדידות ← הגשה לוועדה ← תגובות והשלמות ← אישור / היתר. לכל שלב משך יעד; צפי סיום כולל על התיק מהיום הראשון.
3. **הוועדה = "גורם חיצוני":** "ממתינים לוועדה עד ___" — הגיע התאריך בלי תשובה ← נדנוד + יומן + עדכון יזום ללקוח שאנחנו רודפים.
4. **יומן תיק ולקוח מעודכן יזום** — כמו בכל המחלקות: כל מעבר שלב ← יומן + עדכון ללקוח.
5. **נכס של לקוח שיכול להפוך לעסקת משקיע** ← רק אחרי "פתוח להצעות"=כן עם הסכמה מתועדת — ואז זה עובר למערכת ההצלבות (לא אצלי).
6. **הכסף חלק מהתיק:** אבן חיוב הושלמה ← משימת חשבונית בו-ביום ← לפיננסי.
